POST api/users/authenticate
------------------------------------------------

This endpoint returns an objec with an authentication token. 

Supported fields:
    "username"<str>: User name.
    "password"<str>: Password.

Examples:

    Send:
        {
            "username": "root",
            "password": "root"
        }
        
    Receive:
        {
            "id": 1,
            "username": "root",

        }
